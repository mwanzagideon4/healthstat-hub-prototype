# HealthStat Hub Prototype

This is a simple React prototype of the HealthStat Hub app.
It includes:
- Dashboard
- New Encounter form
- Encounters list
- Analytics: Top Conditions, Medications, Age Bands

## Run Online via StackBlitz
1. Go to https://stackblitz.com/fork/github/<your-repo-url>
2. Wait for StackBlitz to load
3. Click on Preview to use the app

## Run Locally
1. Clone the repository
2. npm install
3. npm start
